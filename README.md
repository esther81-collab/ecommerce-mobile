# Ecommerce Mobile App - Fullstack

A comprehensive fullstack ecommerce mobile application with React Native frontend and Node.js/Express backend.

## Project Structure

```
ecommerce-mobile/
├── mobile/           # React Native mobile app
├── backend/          # Node.js/Express API server
├── docs/             # Documentation
└── docker-compose.yml # Docker configuration
```

## Tech Stack

### Mobile
- **Framework**: React Native with Expo
- **State Management**: Redux Toolkit
- **API Client**: Axios
- **Navigation**: React Navigation
- **UI Library**: React Native Paper

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: PostgreSQL
- **Authentication**: JWT
- **ORM**: Sequelize

## Quick Start

### Backend Setup
```bash
cd backend
npm install
npm run dev
```

### Mobile Setup
```bash
cd mobile
npm install
npm start
```

## Features
- ✅ User Authentication with JWT
- ✅ Product Catalog
- ✅ Shopping Cart
- ✅ Order Management
- ✅ User Profiles
